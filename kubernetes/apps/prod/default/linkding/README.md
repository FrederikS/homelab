# linkding

Self-hosted bookmark manager that is minimal, fast, and easy to set up.

## Dependencies

- Rook Ceph (for persistent storage)

## Configuration

- Internal access only via `linkding.${SECRET_DOMAIN}`
- Uses SQLite database stored in persistent volume
- Default port: 9090
